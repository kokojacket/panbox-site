# Changelog

本文件记录 `panbox-site` 的站点级变更（落地页与文档内容结构、主题与构建配置等）。

## [0.2.0] - 2026-07-21

- 重构站点导航、侧边栏、页面目录与中文搜索界面，统一首页和文档页的主题与响应式布局
- 更新首页产品预览与订阅区域，加入 Search、Autosave、Sync 静态 Demo、桌面版下载入口，并修复套餐按钮对齐
- 新增从零部署路线、桌面版下载和腾讯元器机器人接入教程，以及配套工作流附件与部署截图
- 扩充服务器购买、Docker 安装、域名解析、NginxWebUI、HTTPS 和三个产品的脚本部署步骤与故障排查
- 同步产品说明、安全基线、使用边界与免责声明，并更新 Search 的 PHP 8.4 / ThinkPHP 8 技术口径

## [0.1.0] - 2026-02-10

- 初始化 VitePress 静态站点（`pnpm` + `vitepress`）
- 完成综合落地页（Hero / 产品卡片 / Quickstart / 风险提示）
- 完成文档信息架构与侧边栏：Search / Autosave / Sync + 通用风险
- 迁移 Sync 截图与 logo 到 `docs/public/`
- 完成主题定制：浅色背景氛围、卡片分区、胶囊 CTA、克制动效
