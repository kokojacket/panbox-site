# PanBox 桌面版下载

无需准备服务器即可先在本机体验 PanBox。当前提供 Autosave 自动转存和 Sync 文件处理桌面版。

::: tip 使用建议
桌面版更适合快速体验；正式、长期使用时，仍建议将 Search、Autosave 和 Sync 全部部署到服务器。服务器带宽、流量额度或磁盘空间不足时，可以把 Sync 留在家庭宽带环境运行。
:::

下载按钮会在新标签页打开夸克网盘。安装包版本与文件内容以分享页面为准。
:::
