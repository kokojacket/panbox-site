# 使用边界与免责声明

PanBox Search、Autosave 和 Sync 是自托管工具。使用者应自行确认资源来源、部署地区和具体使用方式符合适用法律法规、网盘平台规则及相关权利人的授权要求。

## 使用责任

- 仅处理自己拥有、已获授权或依法可以使用的内容。
- 不得利用 PanBox 传播违法违规内容、侵犯他人权益或绕过平台安全机制。
- 网站备案、域名实名、服务器安全和账号管理由实际部署者负责。

## 服务与资源稳定性

- 网盘接口、平台规则、限流策略和分享链接状态可能随时变化。
- PanBox 可以减少部分重复操作和单一网盘依赖，但不能保证资源永不失效、账号永不受限或服务持续可用。
- “降低屏蔽影响”“提升稳定性”等描述是产品目标，不构成结果保证。

## 数据与凭据

- Cookie、Token、密码和网盘账号均属于敏感信息，不要提交到公开仓库或发送给无关人员。
- 更新、迁移、增加磁盘或卸载前应备份数据库、配置和任务数据。
- Sync 脚本的卸载操作会删除本地安装目录及其中数据，执行前必须确认备份可恢复。

## 上线前建议

先使用少量非关键资源验证搜索、转存、同步、文件处理和恢复流程，再逐步增加任务量。生产环境应同时遵循[安全基线](/risk/security-baseline)。

本页用于说明产品使用边界，不替代专业法律、安全或合规意见。
