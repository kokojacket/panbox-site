# 免责声明

本页面用于统一说明 PanBox 系列项目的通用风险与使用边界。

## 使用边界

- 本站内容仅用于技术交流与自托管部署参考，不构成任何法律意见或合规保证。
- 使用者应自行确保行为符合所在地法律法规、服务条款以及相关平台规则。

## 数据与凭据

- Cookie / Token / 密码 / API Key 都属于**敏感凭据**。
- 建议：只在受控网络内访问；启用 HTTPS；为反向代理增加登录或 IP 白名单；定期轮换凭据。

## 联网依赖

- 安装脚本可能会联网下载文件（并可能使用代理镜像源）。建议先下载到本地审阅后再执行。
- 若某些产品存在 License/心跳/远程服务依赖，可能导致在网络受限环境下功能受影响。请以各产品“风险提示”页为准。

## 高权限部署（docker.sock）

- 任何把宿主机的 <code>/var/run/docker.sock</code> 挂载进容器的方案，都等同于给予容器**宿主机级别**控制权限。
- 若必须使用：请在隔离的主机/集群中运行，限制可访问网络与镜像来源，并做好审计与备份。

## 责任限制
