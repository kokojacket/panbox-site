# 风险提示（Autosave）

## 一键脚本联网行为

- 安装脚本会联网下载文件，并可能使用代理镜像源。
- 建议先下载到本地审阅再执行。

## 默认数据库账号密码

- 示例中默认数据库名/用户名/密码均为 `panbox-autosave`。
- 生产环境必须修改，避免被扫库或撞库。

## 敏感凭据

- Cookie / Token 等凭据属于高敏信息，建议只在内网部署，并配合 HTTPS 与访问控制。

## License / 联网依赖

- Compose 中包含 License 配置（Base URL 与 Product Name）。
- 若部署环境网络受限，可能导致功能受影响；请先评估。
